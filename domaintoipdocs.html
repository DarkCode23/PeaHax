Domain To IP API Documentation

Our Domain To IP API will return the IP address associated with a domain. This API is optimized for blazing fast response time.


Calling the API

In order to call our Domain To IP API you must send request to this URL
https://www.ps3cfw.com/apis/domaintoip.php


Parameters

Our Domain To IP API has 2 parameters
DOMAIN (required)
API_KEY (required)

Here is an example of a valid call to our Domain To IP API
https://www.ps3cfw.com/apis/domaintoip.php?DOMAIN=ps3cfw.com&API_KEY=your_api_key


Responses

Our Domain To IP API responds in a JSON format.

An example of a successful response from our Domain To IP API Would look similar to this
{
    "status":"success",
    "domain":"ps3cfw.com",
    "ip":"107.180.50.236"
}
