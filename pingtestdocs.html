Ping Test API Documentation

Our Ping Test API will allow you to 'ping' an IP and receive a response that will include a TTL (Time To Live) value coming from the system that you've sent ICMP request packets to. This API is optimized for blazing fast response time.


Calling the API

In order to call our Ping Test API you must send request to this URL
https://www.ps3cfw.com/apis/pingtest.php


Parameters

Our Ping Test API has 3 parameters
IP (required)
API_KEY (required)
FORMAT (optional)

Set FORMAT to 0 for a Raw Text response format, or set FORMAT to 1 for JSON response format

Here is an example of a valid call to our Ping Test API
https://www.ps3cfw.com/apis/pingtest.php?IP=107.180.50.236&API_KEY=your_api_key&FORMAT=0


Responses

An example of a successful response from our Ping Test API Would look similar to this if FORMAT was set to 0
PING 107.180.50.236 (107.180.50.236) 56(84) bytes of data.
64 bytes from 107.180.50.236: icmp_seq=1 ttl=64 time=0.050 ms
64 bytes from 107.180.50.236: icmp_seq=2 ttl=64 time=0.041 ms
64 bytes from 107.180.50.236: icmp_seq=3 ttl=64 time=0.044 ms
64 bytes from 107.180.50.236: icmp_seq=4 ttl=64 time=0.047 ms

--- 107.180.50.236 ping statistics ---
4 packets transmitted, 4 received, 0% packet loss, time 3000ms
rtt min/avg/max/mdev = 0.041/0.045/0.050/0.007 ms

Or for a JSON response format set FORMAT to 1
