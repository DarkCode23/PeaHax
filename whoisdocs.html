WHOIS Lookup API Documentation

Our Whois Lookup API allows you to view ownership records for any valid domain name. This API is optimized for blazing fast response time.


Calling the API

In order to call our WHOIS Lookup API you must send request to this URL
https://www.ps3cfw.com/apis/whoislookup.php


Parameters

Our WHOIS Lookup API has 3 parameters
DOMAIN (required)
API_KEY (required)
FORMAT (optional)

Set FORMAT to 0 for a Raw Text response format, or set FORMAT to 1 for JSON response format

Here is an example of a valid call to our WHOIS Lookup API
https://www.ps3cfw.com/apis/whoislookup.php?DOMAIN=ps3cfw.com&API_KEY=your_api_key&FORMAT=0
